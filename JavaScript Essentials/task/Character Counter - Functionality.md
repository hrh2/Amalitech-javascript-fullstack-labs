# JavaScript Essentials: Character Counter
## Learning Objectives

* **DOM Manipulation:** Dynamically update content based on user interactions.
* **Event Handling:** Implement event listeners to update the character count in real time.
* **String Methods:** Use JavaScript string properties and methods to calculate character length.
* **User Experience Enhancement:** Provide dynamic visual feedback, such as warnings when approaching a character limit or displaying reading time estimates.

## Introduction

In this lab session, you will add functionality to your UI built in Lab 1. You will write JavaScript to update the character count in real time as users type, along with any additional features (e.g., warnings when limits are exceeded or an approximate reading time). The evaluation for this lab will be conducted via peer review with a clear, detailed rubric.

## Project Setup

1. **Use the UI from Lab 1:** Start with the project files from the first lab.
2. **Review JavaScript concepts:** Brush up on event handling, string manipulation, and DOM updates.

## Tasks

### 1. JavaScript Functionality

* **Event Handling:** Attach an event listener to the text field to monitor changes.
* **String Length Calculation:** Use string methods to count characters, words, and sentences.
* **DOM Manipulation:** Dynamically update the character count and display it as the user types.
* **UI Feedback:** Implement warnings for exceeding character limits or show an estimated reading time.

### 2. Testing and Refinement

* Test that the character count updates accurately in real time.
* Ensure any additional features work correctly.
* Refine the code based on your own tests and peer feedback.
